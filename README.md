# Система фінансової звітності (Financial Reporting System)

Професійний веб-застосунок на базі **Laravel 12** та **Filament 5** для управління та аналізу фінансової звітності підприємств. Реалізує показники платоспроможності та нечітку модель оцінки кредитоспроможності.

---

## 🚀 Основні можливості

- **Адмін-панель Filament** — сучасний інтерфейс для управління фінансовими звітами
- **Форма №1 (Баланс)** — введення статей балансу звітного стану підприємства
- **Форма №2 (Фінансові результати)** — введення статей звіту про прибутки та збитки
- **Показники платоспроможності** — автоматичний розрахунок 11 коефіцієнтів по групах G₁, G₂, G₃
- **Нечітка модель кредитоспроможності** — 6-крокова оцінка m(P) з рейтингом I–V категорії та налаштуванням вагів ОПР
- **Друкована форма** — повний звіт з таблицями балансів, показниками та результатами оцінки

## 📊 Аналіз фінансових показників

Показники розраховуються автоматично на основі даних форм №1 та №2.

### Група G₁ — Фінансова стійкість
| Коефіцієнт | Формула | Норма |
|---|---|---|
| K₁ Миттєва ліквідність | (1160 + 1165) / 1695 | > 0,2 |
| K₂ Поточна ліквідність | (1125 + 1155 + 1160 + 1165) / 1695 | > 0,5 |
| K₃ Загальна ліквідність | 1195 / 1695 | > 1,0 |
| K₄ Фінансова незалежність | (1525 + 1595 + 1695) / 1495 | < 1,0 |
| K₅ Маневреність коштів | (1495 − 1095) / 1495 | > 0,1 |

### Група G₂ — Прибутки та збитки
| Коефіцієнт | Формула | Норма |
|---|---|---|
| K₆ Рентабельність виробництва | 2350 / (2500+2505+2510+2515+2520) | > 0,05 |
| K₇ Діяльність минулих років | Градація (0;5] | > 3 |
| K₈ Повернення кредиту | Sₖ / S | ≥ 1,0 |

### Група G₃ — Ефективність управління
| Коефіцієнт | Формула | Норма |
|---|---|---|
| K₉ Термін існування | Роки функціонування | ≥ 3 р. |
| K₁₀ Питома вага коштів | К / S | ≥ 0,3 |
| K₁₁ Ліквідне майно | М / S | ≥ 1,0 |

## 🔍 Нечітка модель оцінки кредитоспроможності

Реалізовано **6-крокову нечітку модель** за загальним алгоритмом, описаним у завданні.

### Алгоритм
1. Вхідні дані V₁–V₂₂ з балансів та додаткових полів
2. Обчислення критеріїв K₁–K₁₁
3. Функції належності μ(K₁)…μ(K₁₁) ∈ [0; 1]
4. Вагові коефіцієнти vᵢ ∈ [1; 10] (задаються ОПР) → нормовані wᵢ
5. Агрегована оцінка: **m(P) = Σ wᵢ · μ(Kᵢ)**
6. Визначення рейтингової категорії

### Рівні кредитоспроможності
| Категорія | Рейтинг | m(P) | Опис |
|---|---|---|---|
| I | ААА / АА | (0,57; 1] | Найвищий рівень |
| II | А / ВВВ | (0,37; 0,57] | Висока кредитоспроможність |
| III | ВВ / В | (0,19; 0,37] | Спекулятивний рейтинг |
| IV | ССС | (0,11; 0,19] | Можливий дефолт |
| V | С / RD / D | [0; 0,11] | Дефолт неминучий |

## ⚙️ Встановлення та налаштування

### 1. Клонування репозиторію
```bash
git clone <url-репозиторію>
cd financial-reporting-system
```

### 2. Встановлення залежностей та налаштування оточення
```bash
composer run setup
```

### 3. Міграція та наповнення бази даних
```bash
php artisan migrate --seed
```
