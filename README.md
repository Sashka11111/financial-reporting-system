# Система фінансової звітності (Financial Reporting System)

Професійний веб-застосунок на базі **Laravel 12** та **Filament 5** для управління та аналізу фінансової звітності (Баланс форми 1-м, 2-м тощо).

## 🚀 Основні можливості

-   **Адмін-панель Filament**: Сучасний інтерфейс для управління даними.
-   **Управління звітами**: Зручний облік фінансових звітів (`FinancialReport`).
-   **Локалізація**: Повна підтримка української мови.

## 🛠 Технологічний стек

-   **Backend**: PHP 8.2+, Laravel 12
-   **Admin Panel**: Filament 5 (TALL stack: Tailwind CSS, Alpine.js, Laravel, Livewire)
-   **Database**: SQLite (за замовчуванням)
-   **Frontend Build Tool**: Vite

## ⚙️ Встановлення та налаштування

### 1. Клонування репозиторію
```bash
git clone <url-репозиторію>
cd financial-reporting-system
```

### 2. Встановлення залежностей та налаштування оточення
Ви можете використати вбудовану команду налаштування:
```bash
composer run setup
```


## 📂 Структура проекту

-   `app/Filament/Resources` — ресурси адмін-панелі для управління фінансовими даними.
-   `data/` — зразки Excel-файлів для імпорту (Баланси форма-1м, 2-м).
-   `database/migrations/` — структура таблиць для звітів, імпортів та експортів.
-  `database/seeders/` — початкові дані для тестування.
